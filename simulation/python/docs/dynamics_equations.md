# 动力学模型

本工程使用一个适合课程仿真的 6 自由度水下机器人模型：

\[
M \dot{\nu} = \tau_{thruster} + \tau_{damping} + \tau_{restoring}
\]

\[
\dot{\eta} = J(\eta)\nu
\]

其中：

- \(\eta=[x,y,z,\phi,\theta,\psi]^T\)：世界坐标位置和 ZYX 欧拉角；
- \(\nu=[u,v,w,p,q,r]^T\)：体系速度；
- \(M\)：刚体质量/转动惯量加上附加质量的对角近似；
- \(\tau_{thruster}\)：五个推进器产生的合力和力矩；
- \(\tau_{damping}\)：线性阻尼和二次阻尼；
- \(\tau_{restoring}\)：重力、浮力、横滚/俯仰回复力矩。

## 推进器分配

第 \(i\) 个推进器位置为 \(r_i\)，单位推力方向为 \(d_i\)，最大推力为 \(T_{max,i}\)，归一化指令为 \(c_i\in[-1,1]\)：

\[
F_i = c_i T_{max,i} d_i
\]

\[
\tau_i = \begin{bmatrix}F_i \\ r_i \times F_i\end{bmatrix}
\]

合力矩为：

\[
\tau_{thruster}=A c
\]

控制器先给出期望广义力：

\[
\tau_d=[F_x,0,F_z,M_x,M_y,M_z]^T
\]

然后用加权最小二乘求解：

\[
\min_c \|W(Ac-\tau_d)\|_2^2, \quad -1\le c_i\le 1
\]

由于该机器人没有侧向推进器，侧向 \(F_y\) 不作为主要控制自由度。

## 控制任务

1. **水下定深前进**：深度 PD + 速度 P + 航向 PD。
2. **转弯**：保持深度和前进速度，同时给定 90° 航向变化。
3. **矩形轨迹追踪**：使用 waypoint guidance，依次跟踪矩形四个边，并用航向控制器指向当前目标点。

该模型不是高精度 CFD，也没有真实推进器曲线；它是一个参数透明、可解释、可运行、适合课程演示和二次开发的动力学仿真基线。
